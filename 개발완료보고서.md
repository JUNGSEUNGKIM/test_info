## 2. 주요 수행 업무

### 2.1 기존 문제점
- 느린 응답 속도 (예: 특정 API 평균 응답 3초 이상)
- 인증 로직 불안정
- DB 트래픽 병목
- 코드 중복 및 모듈화 부족

### 2.2 개선 작업 내역

- **성능 개선**
  - 비동기 처리 도입 (`Promise.all`, Queue 사용 등)
  - 쿼리 최적화 / 인덱스 추가
  - Redis 캐싱 전략 도입

- **API 개선**
  - RESTful 구조 정리
  - Swagger 문서화 및 자동화
  - 공통 응답 포맷 정의

- **DB 리팩토링**
  - 정규화 / 테이블 재설계
  - 조회 성능 향상을 위한 인덱싱
  - 불필요한 컬럼 제거

- **보안 강화**
  - JWT 만료 처리 및 Refresh 토큰 로직 개선
  - 사용자 비밀번호 bcrypt 암호화

- **CI/CD 구축**
  - GitHub Actions + Docker Compose + Nginx 기반 자동 배포 구축
